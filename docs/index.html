一键安装

四种方式，装到的东西完全一样。选你顺手的。

npm i -g @angleschina/angles && angles install

支持 Linux · macOS · Windows · WSL2 · 树莓派 · Alpine

为什么选 Angles

不是又一个 wrapper，是真正能干活的终端 agent。
