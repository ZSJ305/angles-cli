常见问题

更多问题见 FAQ 页面。

API Key 存在哪里？会上传吗？
Key 存储在本地 ~/.angles/config.json，也可用环境变量 ANGLES_API_KEY。请求直接发往你配置的模型供应商，不经过任何中转服务器。
没有 Rust 能用吗？
可以。安装器会拉取预编译二进制（angles-<os>-<arch>.tar.gz）；只有预编译不可用时才 fallback 到源码编译——此时才需要 Rust 工具链，安装器也会自动装好。
支持哪些模型？能用本地 Ollama 吗？
11 家预置（OpenAI / Claude / Gemini / DeepSeek / Grok / Qwen / GLM / Kimi / MiniMax / OpenRouter / 自定义）。任何 OpenAI 兼容端点（含本地 Ollama、vLLM、LM Studio）选「自定义」填 http://localhost:11434/v1 即可。
Windows 原生支持吗？
支持。PowerShell 一行安装：irm https://zsj305.github.io/angles-cli/install.ps1 | iex。也支持 WSL2 中的 bash 脚本。
和 Codex CLI / Claude Code 有什么区别？
Angles 是 Rust 单二进制，零运行时依赖；30 个统一的 angles-* 工具族覆盖文件/目录/终端/Git/网络；11 家供应商一个 CLI 切换。轻量、跨平台、无包装。
