FAQ

More questions? See the FAQ page.

Where is my API key stored? Is it uploaded?
Keys are stored locally in ~/.angles/config.json, or via the ANGLES_API_KEY env var. Requests go directly to your configured provider — no relay server.
Do I need Rust installed?
No. The installer fetches a prebuilt binary (angles-<os>-<arch>.tar.gz). Only if unavailable does it fall back to compiling from source — the installer handles Rust setup automatically.
Which models? Can I use local Ollama?
11 presets (OpenAI / Claude / Gemini / DeepSeek / Grok / Qwen / GLM / Kimi / MiniMax / OpenRouter / Custom). Any OpenAI-compatible endpoint (Ollama, vLLM, LM Studio) — select "Custom" and enter http://localhost:11434/v1.
Does it work natively on Windows?
Yes. One-line PowerShell install: irm https://zsj305.github.io/angles-cli/install.ps1 | iex. WSL2 with bash is also supported.
How does it differ from Codex CLI / Claude Code?
Angles is a single Rust binary with zero runtime deps. 30+ unified angles-* tools cover file/dir/terminal/Git/web. 11 providers, one CLI. Lightweight, cross-platform, no bloat.
